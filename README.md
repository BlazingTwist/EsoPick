## Steps to use

1. Run with: `java -jar <aya_dir>/Aya.jar <main_dir>/ <main_dir>/main.aya`  
  `<aya_dir>` should contain `Aya.jar`, `ayarc.aya`, `base`, `std` and `test`  
  `<main_dir>` should contain `main.aya`, `utils` and `scenes` (and also this readme)
